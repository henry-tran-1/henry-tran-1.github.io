JS Fundamentals

Relationship of Javascript to HTML and CSS
These three languages are used to write code that work together to create a website page that has content (HTML), looks great (CSS), and is interactive (Javascript).
An analogy for these three components would be if a website was a car.

  • The HTML would be the frame and structure of the car.
  • CSS would be the paint, logo and other styling of the car.
  • And Javascript would be the engine, wheels and other parts that make the car run.

Control flow and Loops
Control flows describes the sequence in which a code is executed by a computer or browser. In programming this generally starts at the very top of the code, and then runs line by line down until the very bottom of the code.
An example of control flow in real life would be following a cooking recipe. A simple recipe for cooking eggs in a pan might be:

  1. Heat up pan
  2. Crack eggs into pan
  3. Serve eggs when cooked
The cooking instructions are meant to be followed in order. If this recipe wasn't executed in the listed sequence, you may end up serving the eggs before you even cook them!

Loops are parts of code that are repeated continually for a set amount of times, or until a certain condition is reached.
An example of a loop in every day life would be cleaning your football boots. The steps for cleaning your boots might be:
  1. Scrub them
  2. Hose them down
  3. Check if they are still dirty
In step 3, if they are still dirty you would repeat the instruction (for the second loop). Only when they are no longer dirty would you exit this loop.


DOM - the Document Object Model
As mentioned above, a webpage is made up of 3 components: HTML, CSS and Javascript. HTML and CSS comprise the content and look of a webpage.
Javascript interacts with these 2 components and makes the webpage interactive by changing the HTML and CSS code.
The DOM represents the webpage in a way that allows a programming language like Javascript to read and make changes to it. The DOM represents the content and structure of a webpage in hierachical structure that is composed of a tree-like shape of connected nodes/elements (see below).

An example of Javascript making a website interactive would be if you wanted to change the colour of box on a webpage after a user clicks a button.
When the user clicks the button (an event), you would use Javascript to search the document object for the element with the 'box' label and then change the code that describes its background colour.

Arrays vs. Objects
Arrays
Arrays are used to store a list of similar items.

let fruits = ["apple", "banana", "cherry"]
This would assign these fruit items to the variable fruits.

Each item has an index, which refers to what number on the list it is.
To access a certain item in the array, you need to use its index number in square bracket notation:

let favouriteFruit = fruits[2]
This assigns cherry to the favouriteFruit variable. (Don't forget indices start counting at 0)

Objects
Objects are used to store data as properties. These properties consist of key-value pairs: a key is the name of a property; and the value is its corresponding value.
For example, if you stored your car as a Javascript object you may include the properties:
let car = {manufacturer: "Toyota", model: "Swift", colour: "White"}]

To access a certain property in the object, you could use dot notation:
let favouriteColour = car.colour
This assigns the colour of the above car to the favouriteColour variable.

Or you could use square bracket notation:
let favouriteColour = car["colour"]
Which does the same as above.

Functions
A function is a sequence of code that can be reused again later in the program. This is helpful because if you know you will need your program to do the same thing again later, you can write the function once and activate (call) this bit of code whenever it is needed.
An example of this in real life would be combining salt and pepper into one shaker. If you knew you always added salt and pepper together to your meal, having one shaker that contained a mixture of both would save you from having to use two shakers separately each time.
